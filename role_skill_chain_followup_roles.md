# 后续角色技能梳理（按 `role_skill_chain` 方法）

> 文档用途：系统录入前的结构化确认稿（可直接继续补充/修订）
> 
> 范围：苍炎魔女、贤者、魔弓、魔枪、灵符师、吟游诗人、勇者、格斗家、圣弓、剑帝、兽灵武士、灵魂术士、月之女神、血之巫女、蝶舞者。

## 1. 梳理口径（与 `role_skill_chain` 对齐）

| 维度 | 说明 |
|---|---|
| 发动前提条件 | 发动前必须满足的资源/状态/牌面条件（含弃牌、移除指示物、形态限制等） |
| 发动时机 | 对应回合/结算时间点（如回合开始、①攻击前、②命中后、③伤害阶段、⑤实际伤害产生后、⑥承受伤害时） |
| 产生效果 | 技能完成后对目标/自己/战绩区/状态造成的变化 |
| UI端改造 | 前端需要新增的校验、弹窗、选牌器、目标选择器、状态展示、互斥锁定等 |
| 专属指示物 | 该技能涉及的角色私有资源/状态（如重生、兽魂、形态、永恒乐章等） |
| 专属技能卡 | 是否依赖独有/专属卡才能发动 |
| 待确认项 | 当前文本存在歧义或缺失实现细节的地方 |
